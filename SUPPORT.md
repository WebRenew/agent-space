# Support

## Where to Get Help

- Documentation: `web/content/docs/` and `/docs` in the web app
- Usage questions: open a GitHub issue with the `question` label
- Bug reports: use the Bug Report issue template
- Feature requests: use the Feature Request issue template

## Security Issues

Do not file public issues for vulnerabilities.
Follow [`SECURITY.md`](./SECURITY.md).

## Maintainer Response

This is an open source project maintained on a best-effort basis.
Please include clear reproduction steps and environment details when reporting issues.
